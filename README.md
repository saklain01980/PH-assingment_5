1️⃣ var vs let vs const

var: The old way to declare variables. Try to avoid it.

let: Use this when the value will change later.

const: Use this when the value won't change.

2️⃣ The Spread Operator (...)
It "unpacks" a box. It takes all the items out of an array or object and spreads them out so you can easily copy them or merge them with other things.

3️⃣ map() vs filter() vs forEach()

map(): Changes every item and gives you a new array (e.g., doubling every number).

filter(): Keeps only the items that pass a test and gives you a new array.

forEach(): Just loops through the items to do something. It doesn’t give you a new array.

4️⃣ Arrow Function
A modern, shorter way to write functions using =>. It saves you from typing the word "function" and makes your code look much cleaner.

5️⃣ Template Literals
Strings with superpowers. By using backticks (`) instead of normal quotes, you can hit "Enter" to make multi-line strings, and easily inject variables right into the text like this: `Hello ${name}!`
